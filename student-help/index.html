학생 도움말

학생 로그인과 출석 인증

처음 로그인부터 위치 권한 설정, 출석 인증까지 순서대로 안내합니다.

로그인 방법관리자 승인위치 권한출석 인증

1. 학생 로그인

학생용 계정은 Google 계정을 사용하지 않습니다. 학교에서 받은 학생 ID와 비밀번호를 사용합니다.

  1. 학생 로그인 페이지를 엽니다.
  2. 학생 ID와 비밀번호를 입력합니다.
  3. 로그인 승인 요청을 누릅니다.

2. 관리자 승인

처음 로그인하는 기기는 관리자의 승인이 필요합니다.

  1. 로그인 요청을 보내면 승인 대기 화면이 나타납니다.
  2. 관리자가 알림에서 학생 이름과 기기를 확인합니다.
  3. 관리자가 승인하면 학생 화면이 자동으로 다음 단계로 이동합니다.
거절되면 학생 ID와 기기를 확인한 뒤 다시 요청하세요.

3. 위치 권한 허용

해강중학교 반경 10m 안에서 출석을 인증하기 위해 위치 권한과 정확한 위치가 필요합니다.

  1. iPhone·iPad: 위치 요청에서 허용을 누르고, 설정에서 정확한 위치를 켭니다.
  2. Android: 앱 사용 중 허용과 정확한 위치 사용을 선택합니다.
  3. PC: 주소창 자물쇠 → 사이트 설정 → 위치 → 허용으로 변경합니다.
웹사이트는 운영체제 권한을 강제로 ‘항상 허용’으로 설정할 수 없습니다. 기기에서 제공하는 가장 넓은 허용 범위를 선택하세요.

4. 위치 출석 인증

학교 도착 후 학생 출석 화면에서 인증합니다.

  1. 위치 확인 및 출석 인증을 누릅니다.
  2. 학교와의 거리 및 GPS 정확도를 서버가 확인합니다.
  3. 범위 안이면 출석이 기록되고 완료 메시지가 표시됩니다.
건물 안에서는 GPS 오차가 커질 수 있습니다. 위치 확인이 실패하면 창가나 야외에서 다시 시도하세요. 실제 좌표는 출석 로그에 저장하지 않고 거리와 정확도만 기록합니다.

5. 자동 로그인

관리자가 승인한 학생 기기는 패스키 없이 계속 사용할 수 있습니다. 학생 토큰은 만료일 없이 유지되며, 관리자가 계정을 비활성화하거나 기기 토큰을 해제하면 다시 승인을 받아야 합니다.
