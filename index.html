학생 등록 시트 바로 열기
  1. 위 버튼을 눌러 Google Sheets의 ‘사용자’ 시트를 엽니다.
    학생 정보는 이 시트에서 관리합니다.
  2. 새 행에 학생 정보를 입력합니다.
    name에는 학생 이름, fingerId에는 지문센서에서 사용할 숫자 ID를 입력합니다.
  3. active를 TRUE로 설정합니다.
    TRUE 사용을 권장합니다. 현재 빈칸도 활성으로 처리되며, FALSE는 출석 대상에서 제외됩니다. memo는 선택 사항입니다.
  4. ESP32 지문 등록 모드에서 같은 fingerId로 지문을 등록합니다.
    시트의 fingerId와 센서의 등록 번호가 반드시 같아야 합니다.
  5. 대시보드로 돌아가 새로고침합니다.
    전체 학생 수가 표시되면 등록이 완료된 것입니다.
사용자 시트 열 순서: name · fingerId · active · memo
fingerId는 1부터 시작하지 않아도 되지만, 다른 학생과 중복되면 안 됩니다.
