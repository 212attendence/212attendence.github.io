학생 계정 설정
  1. Apps Script에 apps-script/student-auth-addon.gs를 추가하고 setupStudentAuth_()를 한 번 실행합니다.
  2. 생성된 학생계정 시트에서 studentId, initialPassword, name, fingerId, active를 입력합니다.
  3. 학생이 처음 로그인하면 초기 비밀번호는 해시로 변환되고 initialPassword 칸은 비워집니다.
  4. 대시보드 알림에서 로그인 요청을 승인하거나 거절합니다.
  5. 대시보드 알림 창에서 학교 안의 현재 위치를 해강중학교 중심점으로 저장합니다.
지문 학생 등록 절차
학생 등록 시트 바로 열기
  1. Google Sheets의 사용자 시트를 엽니다.
  2. name, fingerId, active, memo를 입력합니다.
  3. ESP32 지문 등록 모드에서 같은 fingerId로 지문을 등록합니다.
  4. 대시보드를 새로고침해 전체 학생 수를 확인합니다.
