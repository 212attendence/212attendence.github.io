학생용 출석 앱 설치

Android는 네이티브 자동 출석 APK를, iPhone·iPad는 홈 화면 학생 웹앱을 사용합니다.

학생 화면
설치 전 필수 과정

학생 로그인 승인 → 사용 기기 선택 → 위치 권한 허용 → 개인정보 필수·선택 동의 순서로 진행합니다.

사용 기기 선택 및 초기 설정

Android 네이티브 앱

  1. Android 앱 다운로드 페이지에서 현재 버전을 확인합니다.
  2. 다운로드하기를 눌러 APK 파일을 받습니다.
  3. 알 수 없는 앱 설치 허용 안내가 나오면 현재 브라우저에 한해 허용합니다.
  4. 앱 설치 후 위치 권한을 항상 허용으로 설정합니다.
  5. 배터리 사용을 제한 없음으로 변경합니다.
기본 다운로드가 실패하면 Expo 백업 페이지로 자동 이동하며 관리자에게 백업 작동 알림이 기록됩니다.

iPhone·iPad Safari

  1. 학생 출석 화면을 Safari에서 엽니다.
  2. 화면 아래 또는 위의 공유 버튼을 누릅니다.
  3. 홈 화면에 추가를 선택합니다.
  4. 이름이 2-12 학생 출석인지 확인하고 추가합니다.
  5. 학교 도착 후 홈 화면 아이콘을 한 번 열어 출석을 시작합니다.

PC

PC에서는 Android 앱 다운로드와 학생 위치 출석을 지원하지 않습니다. 관리자 대시보드 또는 학생 정보 확인 용도로만 사용하세요.
위치 권한은 출석 기능의 필수 권한입니다. 알림, 오류 진단, 앱 업데이트 안내, 로그인 유지와 Android 백그라운드 위치는 각각 선택할 수 있습니다.
