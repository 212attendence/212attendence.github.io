학생용 앱 설치

학생 출석 앱을 홈 화면이나 시작 메뉴에 설치하면 로그인 화면을 거치지 않고 학생 출석 화면을 빠르게 열 수 있습니다.

학생 화면
자동 설치 버튼

Chrome·Edge 등 설치를 지원하는 브라우저에서는 아래 버튼으로 학생용 앱 설치를 시작할 수 있습니다.

iPhone·iPad Safari

  1. 이 페이지를 Safari에서 엽니다.
  2. 화면 아래 또는 위의 공유 버튼을 누릅니다.
  3. 홈 화면에 추가를 선택합니다.
  4. 이름이 2-12 학생 출석인지 확인하고 추가합니다.

Android Chrome

  1. 오른쪽 위 메뉴 ⋮를 누릅니다.
  2. 앱 설치 또는 홈 화면에 추가를 선택합니다.
  3. 설치를 누른 뒤 홈 화면의 학생 출석 아이콘을 실행합니다.

Windows Chrome·Edge

  1. 주소창 오른쪽의 설치 아이콘을 누릅니다.
  2. 아이콘이 없으면 브라우저 메뉴에서 앱 설치를 선택합니다.
  3. 설치 후 시작 메뉴 또는 바탕화면에서 실행합니다.
학생용 앱과 관리자용 앱은 별도 앱입니다. 이 페이지에서는 반드시 학생용 앱을 설치하세요. 기존 통합 앱이 남아 있다면 삭제한 뒤 다시 설치하는 것이 안전합니다.
