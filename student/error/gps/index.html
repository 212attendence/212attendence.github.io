위치 권한이 설정되지 않았습니다.

학생 출석 인증을 위해 위치 권한과 정확한 위치가 필요합니다.

  1. iPhone·iPad
    설정 → 개인정보 보호 및 보안 → 위치 서비스 → Safari 웹사이트 또는 설치한 출석 앱 → 앱을 사용하는 동안 허용 → 정확한 위치 켜기
  2. Android
    설정 → 위치 → 앱 권한 → Chrome 또는 출석 앱 → 앱 사용 중에만 허용 → 정확한 위치 사용
  3. PC
    브라우저 주소창의 자물쇠 아이콘 → 사이트 설정 → 위치 → 허용
웹사이트는 운영체제의 위치 권한을 강제로 ‘항상 허용’으로 바꿀 수 없습니다. 표시되는 가장 넓은 허용 범위를 선택하고 정확한 위치를 켜야 합니다.
권한 설정 후 다시 시도 권한 설정 도움말로 이동
